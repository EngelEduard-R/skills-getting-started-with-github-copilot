# Agent Instructions for this repository

This file provides concise, actionable guidance for AI coding agents working in this repository.

Keep this file minimal — link to project docs for details.

## Quick commands
- Install dependencies: `pip install -r requirements.txt`
- Run app (development): `python src/app.py`  — serves FastAPI on http://localhost:8000
- Run tests: `pytest` (project uses `pytest.ini`)

## What this project is
- A small FastAPI application serving an in-memory activities API. See [src/README.md](src/README.md) for details.

## Key files and directories
- `src/app.py`: main FastAPI application and routes.
- `src/static/`: frontend assets served at `/static` (index.html, styles, JS).
- `requirements.txt`: Python dependencies.
- `pytest.ini`: pytest configuration.

## Agent guidance and conventions
- Prefer making minimal, focused changes. Keep behavior consistent with in-memory storage (no persistence).
- Link to existing docs rather than copying them. See [README.md](README.md) and [src/README.md](src/README.md).
- When adding tests, update `requirements.txt` if new test deps are required.
- Avoid changing public API routes without updating `src/README.md` and the OpenAPI docs generated by FastAPI.

## Common tasks
- Add an endpoint: modify `src/app.py` and add unit tests under `tests/`.
- Improve static UI: edit `src/static/*` and ensure `src/app.py` still mounts the static directory.
